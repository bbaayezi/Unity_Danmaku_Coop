# Unity_Danmaku

这里是一个使用Unity制作的弹幕游戏的项目仓库

## 写在前面

本仓库托管的胆码主要供个人学习使用。由于本人深感个人能力的不足，因此想到叫上一些一同学习Unity的小伙伴一起完成这个项目，同时也是为了更加熟悉一个团队开发的流程。

## 目标

本项目的最终目标是制作出类似于东方系列的弹幕游戏，能够有一个比较完整的试玩体验即可。

## 短期计划

一开始的话是打算先在团队之内做好各种协定和文档的编写，包括C#接口的编写，基类以及抽象类的编写。游戏开发架构确立，分工等等。

## 正在做...

目前正在进行的是第一阶段，也就是团队协定和标准的确立。目前打算采用的方法是，每位成员独立创建一个branch，如果遇到有意见分歧，或者是没有办法及时与其他成员进行讨论的时候，先在各自的分支上进行代码的修改，之后再定期讨论决定采用哪种方案，并合并到master分支。

## 具体任务列表

- [ ] 游戏整体开发架构确立
- [ ] 自机功能开发的文档和标准确立
- [ ] 自机基础功能接口开发，包括移动，射击，放B，低速模式等
- [ ] 自机子弹发射功能接口开发
